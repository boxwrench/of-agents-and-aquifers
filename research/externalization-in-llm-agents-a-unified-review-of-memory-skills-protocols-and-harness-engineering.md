# Externalization in LLM Agents: A Unified Review of Memory, Skills, Protocols and Harness Engineering

> [!NOTE]
> **Source:** [arXiv](https://arxiv.org/abs/2604.08224)
> **Date Added:** 2026-06-08
> **Published:** 2026-04-09

## Summary
This technical report reviews how LLM agent capabilities are increasingly externalized into memory stores, reusable skills, interaction protocols, and harness engineering rather than being handled only through model weights or prompt context. It frames memory, skills, protocols, and harnesses as cognitive infrastructure that makes agent behavior more reliable, governable, and reusable.

## 💧 Relevance to Of Agents and Aquifers

- Useful for water and infrastructure workflows where reliability depends on governed execution, auditable memory, reusable domain skills, and protocol-aware integration with telemetry or modeling tools.
